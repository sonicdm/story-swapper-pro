Story Swapper

Mad Libs-style word swap fun

Opened as a local file — book downloads need a web server. From this folder run npm run dev or npm run preview, or serve the built app: cd dist && python -m http.server 8080 then open http://localhost:8080/. Sample and Paste work offline.

    Classic fill-in-the-blank stories from madlibs-api (madlibz templates, MIT). Also accepts Rosetta-style <noun> and --NOUN-- syntax when you paste.
